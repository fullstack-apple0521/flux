## gotk suspend alert

Suspend reconciliation of Alert

### Synopsis

The suspend command disables the reconciliation of a Alert resource.

```
gotk suspend alert [name] [flags]
```

### Examples

```
  # Suspend reconciliation for an existing Alert
  gotk suspend alert main

```
