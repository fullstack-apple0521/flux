## flux delete alert

Delete a Alert resource

### Synopsis

The delete alert command removes the given Alert from the cluster.

```
flux delete alert [name] [flags]
```

### Examples

```
  # Delete an Alert and the Kubernetes resources created by it
  flux delete alert main
```
